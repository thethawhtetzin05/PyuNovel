# 🧞‍♂️ PyuNovel အသေးစိတ်အသုံးပြုနည်း လမ်းညွှန် (Detailed Step-by-Step Guide)

ဒီလမ်းညွှန်မှာ PyuNovel ရဲ့ လုပ်ဆောင်ချက်တစ်ခုချင်းစီကို အစကနေ အဆုံးထိ ဘယ်လိုအသုံးပြုရမလဲဆိုတာ အသေးစိတ် ရှင်းပြပေးထားပါတယ်။

---

## ၁။ အကောင့်ဝင်ခြင်းနှင့် အကောင့်ဖွင့်ခြင်း (Authentication)

PyuNovel ရဲ့ အင်္ဂါရပ်အပြည့်အစုံ (ဥပမာ - ဝတ္ထုသိမ်းဆည်းခြင်း၊ Check-in ဝင်ခြင်း) ကို အသုံးပြုနိုင်ဖို့ အရင်ဆုံး အကောင့်ဝင်ဖို့ လိုအပ်ပါတယ်။

1.  **Sign In သို့ သွားရန်:** Navbar (မျက်နှာပြင်အထက်ပိုင်း) ရှိ **"Sign In"** ခလုတ်ကို နှိပ်ပါ။
2.  **အကောင့်မရှိသေးပါက:** "Don't have an account?" အောက်ရှိ **"Sign up"** ကို နှိပ်ပြီး နာမည်၊ အီးမေးလ်နှင့် စကားဝှက်တို့ဖြင့် အကောင့်သစ် ဖွင့်ပါ။
3.  **အကောင့်ဝင်ရန်:** အကောင့်ရှိပြီးသားဖြစ်ပါက အီးမေးလ်နှင့် စကားဝှက်ရိုက်ထည့်ကာ **"Sign In"** ကို နှိပ်ပါ။

---

## ၂။ ဝတ္ထုများ ရှာဖွေဖတ်ရှုခြင်း (Browsing & Reading)

**ဝတ္ထုရှာဖွေနည်း:**
1.  **Home (ပင်မစာမျက်နှာ):** App ကို ဖွင့်လိုက်တာနဲ့ နာမည်ကြီးဝတ္ထုတွေနဲ့ နောက်ဆုံးထွက်အခန်းတွေကို မြင်ရမှာဖြစ်ပါတယ်။
2.  **Ranking (အဆင့်သတ်မှတ်ချက်):** Navbar ရှိ **"Ranking"** ကို နှိပ်ပါ။
    *   **Dropdown အသုံးပြုနည်း:** ညာဘက်အပေါ်ရှိ Dropdown (Popular လို့ ရေးထားတဲ့နေရာ) ကို နှိပ်ပြီး -
        *   **Popular:** လူကြည့်အများဆုံးကြည့်ရန်
        *   **Collection:** လူကြိုက်အများဆုံးကြည့်ရန်
        *   **New Release:** အသစ်ထွက်ကြည့်ရန်
        *   **Recently Updated:** နောက်ဆုံးအပ်ဒိတ်ဖြစ်တာတွေကြည့်ရန် စသည်ဖြင့် ရွေးချယ်နိုင်ပါတယ်။

**စာဖတ်နည်း (Reading Guide):**

သင်နှစ်သက်သော ဝတ္ထုမျက်နှာဖုံးကို နှိပ်ပြီး ဝတ္ထုအသေးစိတ်စာမျက်နှာရှိ **"📖 Read Now"** ကို နှိပ်ကာ စတင်ဖတ်ရှုနိုင်ပါတယ်။

### 📱 ဖုန်းဖြင့်ဖတ်ရှုသူများအတွက် (Mobile/App View)
1.  **လွယ်ကူသော Navigation:** အခန်းရဲ့ အောက်ဆုံးတွင် **Prev**, **Contents**, **Next** အပြာရောင်ခလုတ်များကို တွေ့ရမှာပါ။
2.  **TOC Sidebar:** **"Contents"** ခလုတ်ကို နှိပ်လိုက်လျှင် **ဘေးဘက်မှ Sidebar လေး** ပွင့်လာမှာ ဖြစ်ပါတယ်။ ဝတ္ထုအသေးစိတ်စာမျက်နှာသို့ ပြန်သွားစရာမလိုဘဲ အခြားအခန်းများကို ဤနေရာမှ တိုက်ရိုက်ရွေးချယ်နိုင်ပါတယ်။
3.  **စာပိုဒ်အလိုက်မှတ်ချက်ပေးခြင်း:** စာပိုဒ်တစ်ခုချင်းစီရဲ့ ဘေးရှိ **💬 icon** လေးကို နှိပ်ပြီး အဲ့ဒီစာပိုဒ်အတွက် သီးသန့် ကွန့်မန်ပေးနိုင်ပါတယ်။
4.  **Quick Comment:** အောက်ဆုံးရှိ **"COMMENT"** ခလုတ်အဝိုင်းလေးကို နှိပ်ပြီး Sidebar မှတစ်ဆင့် အခန်းတစ်ခုလုံးအတွက် ဆွေးနွေးနိုင်ပါတယ်။

### 💻 ကွန်ပျူတာဖြင့်ဖတ်ရှုသူများအတွက် (Desktop/Web View)
1.  **Navigation:** အောက်ဆုံးရှိ **Prev**, **Contents**, **Next** ခလုတ်များကို အသုံးပြုနိုင်ပါတယ်။
2.  **Table of Contents:** **"Contents"** ခလုတ်ကို နှိပ်လိုက်လျှင် ဝတ္ထုအသေးစိတ်စာမျက်နှာရှိ **Chapters Tab** သို့ တိုက်ရိုက် ပြန်လည်ရောက်ရှိသွားမှာ ဖြစ်ပါတယ်။
3.  **Deep Linking:** Reader မှတစ်ဆင့် ပြန်လာသည့်အတွက် Chapters Tab ကို အလိုအလျောက် ပွင့်လျက်သား မြင်တွေ့ရမှာ ဖြစ်ပါတယ်။
4.  **Wide View:** Desktop view တွင် စာသားများကို ပိုမိုကျယ်ပြန့်စွာ ဖတ်ရှုနိုင်သလို၊ Sidebar အသုံးပြုပုံမှာလည်း ပိုမိုကျယ်ဝန်းတဲ့ အမြင်ကို ရရှိမှာဖြစ်ပါတယ်။

---

## ၃။ ဝတ္ထုများ စုဆောင်းခြင်း (My Collection)

ဝတ္ထုတွေကို နောက်မှ ပြန်ဖတ်ဖို့ သိမ်းထားနည်း-
1.  ဝတ္ထုအသေးစိတ်စာမျက်နှာသို့ သွားပါ။
2.  **Collect ခလုတ်:** Read Now ဘေးရှိ **"🔖 Collect"** ခလုတ်ကို နှိပ်ပါ။ ခလုတ်အရောင် "Collected" လို့ ပြောင်းသွားရင် သိမ်းဆည်းပြီးပါပြီ။
3.  **ပြန်ကြည့်ရန်:** Navbar ရှိ **"Collection"** Menu ကို နှိပ်လိုက်ရင် သင်သိမ်းထားတဲ့ ဝတ္ထုအားလုံးကို စနစ်တကျ မြင်ရမှာဖြစ်ပါတယ်။

---

## ၄။ နေ့စဉ်ဆုလာဘ်များ ရယူခြင်း (Daily Check-in & Level)

1.  **Profile သို့ သွားရန်:** Navbar ရှိ မိမိရဲ့ Avatar (သို့မဟုတ်) Initial နာမည်ကို နှိပ်ပြီး **"Profile"** ကို ရွေးပါ။
2.  **Check-in ဝင်ရန်:** ပရိုဖိုင်စာမျက်နှာအောက်ပိုင်းရှိ **"Daily Check-in"** ခလုတ်ကို နှိပ်ပါ။
3.  **ဆုလာဘ်:** ခလုတ်နှိပ်လိုက်တာနဲ့ သင်ရရှိမယ့် **EXP (+10 EXP စသဖြင့်)** ကို ချက်ချင်းမြင်ရမှာပါ။
4.  **Streak:** နေ့တိုင်းဆက်တိုက်ဝင်လေလေ၊ သင့်ရဲ့ Level က ပိုမြန်မြန်တက်လေလေ ဖြစ်မှာပါ။

---

## ၅။ စာရေးသူများအတွက် ဝတ္ထုတင်နည်း (For Writers)

စာမူစတင်တင်လိုပါက အောက်ပါအဆင့်အတိုင်း လုပ်ပါ-

**ဝတ္ထု (Novel) အသစ်ဆောက်ခြင်း:**
1.  Navbar ရှိ **"Create"** Button ကို နှိပ်ပါ။
2.  ဝတ္ထုခေါင်းစဉ်၊ Cover ပုံ၊ တဂ်များ (Action, Fantasy စသဖြင့်) နှင့် ဇာတ်လမ်းအကျဉ်းတို့ကို ဖြည့်စွက်ပြီး **"Create Novel"** ကို နှိပ်ပါ။

**အခန်းများ (Chapters) ထည့်သွင်းခြင်း:**
1.  မိမိဆောက်ထားသော ဝတ္ထုထဲရှိ **"Add Chapter"** ကို နှိပ်ပါ။
2.  **Volume (အတွဲ) စီမံခြင်း:** အခန်းတွေကို အတွဲလိုက်ခွဲချင်ရင် **"Add New Volume"** ကို အရင်နှိပ်ပြီး အတွဲနာမည်ပေးပါ။
3.  **စာမူတင်ရန်:** အခန်းခေါင်းစဉ်ပေးပါ၊ စာမူများကို Content box ထဲမှာ ရိုက်ထည့်ပါ (သို့မဟုတ်) Copy ကူးထည့်ပါ။
4.  **Publish:** အားလုံးပြီးရင် **"Publish"** ကို နှိပ်ပြီး တင်လိုက်ပါ။

## ၆။ တယ်လီဂရမ် Bot နှင့် ချိတ်ဆက်အသုံးပြုခြင်း

Web ဆိုက်ထဲဝင်စရာမလိုဘဲ Telegram ကနေ စာမူတင်ချင်ရင်-
1.  Navbar ရှိ **"Integration"** သို့ သွားပါ။
2.  **"Generate Connection Link"** ကို နှိပ်ပြီး ကုဒ်နံပါတ် ၆ လုံးကို ရယူပါ။
3.  PyuNovel Telegram Bot ထံသို့သွားပြီး ထိုကုဒ်ကို ပို့ပေးကာ အကောင့်ချိတ်ဆက်လိုက်ပါ။
4.  ချိတ်ဆက်ပြီးပါက Telegram မှတစ်ဆင့် စာသား (Text) သို့မဟုတ် ဖိုင် (Files) ပို့ရုံဖြင့် ဝတ္ထုအခန်းများ အလိုအလျောက် တင်နိုင်မှာဖြစ်ပါတယ်။
